🚗 Auto World

A modern automotive web platform designed to explore cars, services, and vehicle-related solutions in one place.
Auto World provides a clean UI experience for browsing cars, services, and automotive features.
🚀 Features
🚘 Car Listings
Browse available cars with details and visuals.
🔧 Automotive Services
Explore services like maintenance, repair, and accessories.
🛒 Marketplace Concept
View cars, parts, and automotive products.
📱 Responsive Design
Works smoothly on desktop and mobile devices.
🎨 Modern UI/UX
Clean layout, smooth animations, and intuitive navigation.

Auto platforms usually combine listings, services, and user-friendly design to simplify the car-buying and servicing experience ()

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & animations
JavaScript (Vanilla JS) – Functionality
